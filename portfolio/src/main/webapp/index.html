List of projects I've worked on:

Crossword Solver

  • I wrote two versions - one using C and the other using the MIPS assembly language.
  • It could take a dictionary and a crossword, and would then return the words that appeared both in the dictionary and the crossword.
  • It found horizontal, vertical and diagonal words - as well as wrap-around words!

Stock Check App

  • A mobile app written for Windows devices, using Xamarin (C# and XAML).
  • Created for the online coffee company, Another Coffee, to help with stock checks.
  • I used open-source barcode scanner software to help the app read product barcodes. It could then connect to the company database to allow live-updates of the stock levels.
  • It had several uses, including: stock checks, updating stock levels after the arrival of deliveries, and simply allowing the user to get more details about a product in the warehouse.
